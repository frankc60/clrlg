# clrlg
Color Logging

##Install
```
$ npm install clrlg
```

##Usage

```const c = require('clrlg');
 
c.lg("default color");

c.lg("red","red");

c.lg("yellow","yellow");

c.lg("blue","blue");

c.lg("green","green");

c.lg("magenta","magenta");

c.lg("grey","grey");

c.lg("white","white");

c.lg("cyan","cyan");

c.lg("fish color","fish");
```
